💻 Dev tools 💻

  • MSKtiled

    • Swift package that offers the capability to turn Tiled maps into SKTileMapNode and related objects. It also offers additional functionality like pathfinding on the tilemap.

  • MoreSpriteKit

    • Swift package that offers additional node types and effects which you can use in combination with SpriteKit.

  • Universal Spritesheet Generator

    • Originally a fork from Guarav0’s awesome generator, I added a lot of new art and it has now evolved into a very popular spritesheet generator using all available character art from the Liberated Pixel Cup.

  • iOS GameplayKit examples

    • Small project that demonstrates some of Apple’s GameplayKit capabilities.
